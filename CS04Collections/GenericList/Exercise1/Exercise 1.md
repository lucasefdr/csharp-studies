# Exercício 1 (Básico): Sistema de Gerenciamento de Notas de Alunos

## Problema:
Você precisa criar um sistema simples para gerenciar notas de alunos. Cada aluno tem um nome e uma lista de notas.

## Requisitos:

- Adicionar alunos (nome e lista de notas).
- Remover alunos pelo nome.
- Calcular a média de notas de um aluno específico.
- Listar todos os alunos e suas médias.
- Encontrar o aluno com a maior média.
